
# RideWise Customer Churn Prediction & Customer Analytics Project
## Project Overview

RideWise is a fictional European mobility technology company providing ride-hailing services across major metropolitan cities. The company serves over 200,000 active customers and facilitates more than 800,000 monthly trips across five major European cities:

London
Berlin
Amsterdam
Barcelona
Milan

Operating in the highly competitive mobility technology industry, RideWise aims to improve customer retention and maximize customer lifetime value through advanced analytics, machine learning, and predictive modeling.

This project develops an end-to-end machine learning solution capable of identifying customers at risk of churning, segmenting customers based on behavioural patterns, and providing actionable business insights through interactive dashboards and API deployment.

## Business Challenge
RideWise currently faces several business challenges:

Approximately 25% quarterly customer churn

Limited understanding of customer behavioural patterns

Low promotion campaign conversion rates

Unclear return on investment (ROI) for promotional campaigns

Manual customer analysis requiring weeks to complete

No predictive system for identifying at-risk customers

Limited real-time customer insights

Lack of an integrated customer analytics platform

Increasing competitive pressure within the mobility sector

## Project Objectives
The primary objectives of this project are to:

Develop customer churn prediction models using machine learning
Build customer segmentation models using clustering techniques
Engineer meaningful behavioural features from customer activity
Improve model transparency through explainable AI techniques
Deploy real-time customer risk scoring
Build an interactive customer analytics dashboard
Develop REST API endpoints for production inference
Create a production-ready machine learning workflow with monitoring capability

## Dataset Overview
The project uses a comprehensive synthetic dataset representing one year of RideWise customer operations.

Dataset Characteristics
200,000+ Customers
800,000+ Trip Records
12 Months of Customer Activity
Behavioural Analytics
Promotional Campaign Data
External Business Factors
